# LazyCoders Portfolio

A team portfolio website built with Node.js, Express, EJS, TailwindCSS, and PostgreSQL.

## Tech Stack

- Node.js
- Express.js
- EJS (templating)
- TailwindCSS
- PostgreSQL

## Features

- Home page with team member profiles
- Projects page showcasing work
- Contact form that saves to database
- Admin panel to manage messages
- Export messages as CSV
- Dark mode toggle
- Mobile responsive
- Visitor counter

## Local Setup

### Prerequisites

- Node.js 18 or higher
- PostgreSQL 14 or higher

### Steps

1. Clone the repository
2. Run `npm install`
3. Create a PostgreSQL database named `lazycoders_db`
4. Run the SQL schema from `database.sql`
5. Create a `.env` file with your database credentials
6. Run `npm start`
7. Visit `http://localhost:3000`
